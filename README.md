=======
 Projeto Casa de Leilões

 Sobre
 Este projeto tem como objetivo desenvolver um sistema para gerenciamento de uma casa de leilões, permitindo o controle de produtos, lances e clientes de forma organizada e segura.

  Funcionalidades
- Cadastro de produtos e clientes  
- Registro de lances em tempo real  
- Controle de status dos leilões  
- Integração com banco de dados MySQL  

  Tecnologias utilizadas
- Java (NetBeans)  
- MySQL (banco de dados)  
- Git e GitHub para versionamento
